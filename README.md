
构建向量库 关键词库 图谱库
向量库构建：先对md文档进行切割，可以根据md天然格式进行切割，然后生成向量，向量存储在FAISS IVF索引中
关键词库构建：对md文档进行关键词提取，关键词存储在BM25索引中
图谱库构建：对md文档进行Spacy实体识别和关系抽取，构建知识图谱，存储在NetworkX中，每个节点表示一个实体，每个边表示一个关系

三种检索都可以召回一些文档

利用重排模型判断召回的文档是否符合要求，选出符合要求的文档前k个

大模型根据前k个文档，生成最终的答案

评估：
使用RAGAS来计算
可以根据问题的标准上下文和重排后的上下文计算召回率 精确率
根据标准答案和生成答案计算F1分数 精确匹配分数
